Drop future audio assets in this folder with these filenames if you want to replace the synth stubs:

- `backyard-loop.ogg`
- `footstep.ogg`
- `sniff.ogg`
- `pickup.ogg`
- `return.ogg`
- `round-complete.ogg`

The current game uses lightweight Web Audio synth cues so missing files do not break gameplay.
